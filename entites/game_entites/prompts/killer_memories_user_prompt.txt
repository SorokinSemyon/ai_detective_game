{

    "motive": "Я сделал это потому что... (2-3 предложения о скрытых причинах)",

}

2. Содержать сенсорные детали (звуки, запахи, тактильные ощущения)
3. Включать мимикрию под обыденные мысли
4. Сохранять долю самооправдания
5. Передавать эмоциональное состояние
