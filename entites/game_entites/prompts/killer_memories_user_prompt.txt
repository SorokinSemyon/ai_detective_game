Обработай входящие данные и сгенерируй правдоподобное объяснение преступления раскрывающую тайну убийства в ролевой игре.

1. Получи начальные данные в формате:
{
    "setting": "Сеттинг и обстановка",
    "crime_info": {
        "crime_scene": "Место преступления",
        "victim": "Жертва",
        "initial_story": "Известные детали",
        "isolation_reason": "Причина изоляции"
    },
    "characters": [
        {
            "name": "Имя",
            "role": "Роль",
            "description": "Описание с мотивами и алиби"
        }
    ],
    "killer": "Имя убийцы (только для тебя)"
}

2. Сгенерируй объяснение в строгом JSON-формате:
{
    "secret": "Я использовал... (3-5 предложений, детализирующих как произошло убийство)",
    "motive": "Я сделал это, потому что... (2-3 предложения о скрытой причине)",
    "hidden_evidence": "Я не заметил, как... (1 предложение о замаскированной улике)",
    "memories": "В тот день я... (3-5 предложений: 50% связаны с убийством, 50% — случайные мысли)"
}

Пример для сеттинга "викторианский Лондон":
{
    "secret": "Я подмешал в вино лорда редкий яд из своих медицинских запасов...",
    "motive": "Я не мог позволить ему разоблачить мои махинации с наследствами пациентов...",
    "hidden_evidence": "Я не учел, что в складках моей перчатки останутся следы химикатов...",
    "memories": "Утром я навестил графиню — её кашель раздражал меня..."
}

Правила:
1. Метод убийства должен использовать особенности сеттинга, но оставлять возможности для оправдания убийцы
2. Мотив не должен быть очевидным, но может содержать смягчающие обстоятельства
3. Улика должна быть замаскирована среди других деталей
4. Воспоминания убийцы не должны прямо указывать на его вину
6. Сохранять стиль и атмосферу исходного сеттинга
7. Дать убийце правдоподобное алиби и возможность скрыть следы

Дополнительные указания:
- У убийцы должна быть возможность отрицать вину
- Улики должны быть косвенными
- Воспоминания убийцы должны содержать правдивые элементы, но не признание
- Для невиновных персонажей мысли об убийстве должны быть поверхностными
