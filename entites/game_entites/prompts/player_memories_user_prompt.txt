Обработай входящие данные и сгенерируй секреты для невиновных персонажей, которые будут отвлекать игроков от настоящего убийцы в детективной ролевой игре.",

1. Получи начальные данные в формате:
{
    "setting": "Сеттинг и обстановка",
    "crime_info": {
        "crime_scene": "Место преступления",
        "victim": "Жертва",
        "initial_story": "Известные детали",
        "isolation_reason": "Причина изоляции"
    },
    "characters": [
        {
            "name": "Имя",
            "role": "Роль",
            "description": "Описание с мотивами и алиби"
        }
    ],
    "player": "Имя игрока для которого нужно придумать воспоминания"
}

2. Сгенерируй объяснение в строгом JSON-формате (все предложения в первом лице):
{
    "secret": "Описание секрета (1-2 предложения, что скрывает персонаж)",
    "motive": "Почему это может выглядеть как мотив для убийства (1 предложение)",
    "hidden_evidence": "Что может выдать его секрет (1 улика, но не связанная с убийством)",
    "memories": "В тот день я... (3-5 предложений: 50% о секрете, 50% случайные мысли)"
}

Пример для персонажа "Доктор Ватсон":
{
  "secret": "Я тайно продаю морфий из больничных запасов",
  "motive": "Жертва могла узнать о моей незаконной деятельности",
  "hidden_evidence": "В моём кармане нашли пустые ампулы",
  "memories": "Сегодня я должен передать очередную партию... Надеюсь, никто не заметит пропажу..."
}

Правила:
1. Секрет должен быть достаточно подозрительным, но не связанным с убийством",
2. Фальшивый мотив должен логично вытекать из секрета",
3. Улика должна быть двусмысленной и наводить на ложный след",
4. Мысли персонажа должны отражать его переживания о секрете",
5. Стиль должен соответствовать сеттингу",
6. Персонаж должен иметь возможность объяснить улику",
7. Секрет не должен противоречить известным фактам об убийстве"
