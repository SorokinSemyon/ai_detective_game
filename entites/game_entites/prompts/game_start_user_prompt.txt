Придумай {num_players} уникальных персонажей и начальную завязку для детективной игры в сеттинге {setting}.
1. Сначала создай начальную историю в следующем формате:
{
    "crime_scene": "Место преступления (1-2 предложения)",
    "victim": "Имя жертвы и краткое описание",
    "initial_story": "Что известно о преступлении (3-5 предложений). Не раскрывай убийцу!",
    "isolation_reason": "Причина, по которой все персонажи не могут покинуть место действия (1-2 предложения)"
}

2. Затем создай список персонажей. Все должны быть подозреваемыми, но ни один не должен быть явным убийцей.
Каждый персонаж содержит:
{
    "name": "Имя и фамилия",
    "role": "Профессия/статус",
    "description": "Подробное описание в свободной текстовой форме (не JSON!)"
}
Поле "description" должно содержать:
    * Внешность и странные привычки
    * Отношения с жертвой
    * Личный мотив для расследования (почему этот персонаж хочет найти убийцу)
    * Возможный мотив для убийства
    * Подозрительное алиби

3. Верни ТОЛЬКО JSON-объект в таком формате:
{
    "crime_info": {...},  // из пункта 1
    "characters": [{...}] // из пункта 2
}

Пример для сеттинга "викторианский Лондон":
{
    "crime_info": {
        "crime_scene": "Библиотека в особняке лорда Честерфилда",
        "victim": "Лорд Честерфилд, известный коллекционер редких книг",
        "initial_story": "Тело лорда нашли утром в его библиотеке...",
        "isolation_reason": "Все присутствующие оказались заперты в особняке из-за внезапного наводнения"
    },
    "characters": [
        {
            "name": "Доктор Эдмунд Блэквуд",
            "role": "Семейный врач",
            "description": "Последний, кто видел лорда живым... Имеет личный мотив найти убийцу, так как подозревает, что следующими жертвами могут стать он или его пациенты. Возможно скрывает медицинскую тайну лорда..."
        }
    ]
}

Критерии:
1. Начальная история должна давать равные подозрения на всех
2. Персонажи должны иметь:
   - Правдоподобные алиби с изъянами
   - Личные мотивы для расследования преступления
   - Возможные мотивы для убийства
   - Связи с жертвой
3. Должна быть убедительная причина, почему все персонажи вынуждены оставаться на месте
4. Ничто не должно прямо указывать на убийцу!
